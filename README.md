# FEMAS - Landing Page

Landing page creada con Vite + React para FEMAS Tecnología Industrial.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Estructura principal

- `index.html` - Punto de entrada
- `src/` - Código fuente
  - `components/` - Componentes React separados
  - `App.jsx` - Componente raíz
  - `App.css` - Estilos globales
